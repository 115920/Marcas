1- Sintaxis CSS

selector { propiedad: valor; }

 Ejemplo:

body { background-color: yellow; }

body { background-color: yellow; color: blue; margin-left: 50%; }

<link type=“text/css” rel=“stylesheet” href=“estilo.css”>

2- Color:

  • Nombre:

    • black, silver, grey, white, maroon, green, lime, olive, yellow, navy, red, purple, fuchsia, blue, teal, aqua...

  • Modelo RGB:

    • – Hexadecimal:

    • #RGB (0-F)

    • #RRGGBB (00-FF)

    – rgb/rgba (0 = transparente; 1 = opaco):

    • Octetos: rgb(rojo, verde, azul) (0-255)

    • Porcentajes: rgb(r%, v%, a%) (0-100)

  • Colores seguros:

    • – Daltonismo

    – 256 colores: 00, 33, 66, 99, CC, FF

    3- Clases:

    selector → selector.clase

    CSS:

    p { color: blue; }

    HTML:

    <p> Este texto es azul </p>

     

    CSS:

    p.verde { color: green; }

    HTML:

    <p class=“verde”> Este texto es verde </p>
